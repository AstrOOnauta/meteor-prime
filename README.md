## Controle de Finanças

## 💻 Linguagens e tecnologias utilizadas

    - React Native
    - Expo
    - Typescript
    - Styles-Components
    - Hooks
    - React-navigation
    - React-native-async-storage
    - React-native-stars
    - The movie DB API
    - Eslint
    - Prettier
    - Yarn

<hr>

## Instalação

    - Clonar repositorio
    $ git clone https://github.com/AstrOOnauta/meteorprime.git

    - Instalar dependências
    $ yarn

    - Rodar aplicativo
    $ expo start
